# afanime-tg

[English Version](./README_en.md)

一个自动转发动画仓库的消息至你自己的Telegram频道&话题的机器人，由 Bangumi & [Real Search](https://search.acgn.es/)提供支持。

## 命令

* `/start` 显示欢迎信息
* `/about` 显示机器人信息
* `/help` 显示帮助信息
* `/dailytask`  手动更新所有在追的动画视频（手动执行日常任务）
* `/getid` 获取当前话题的ID（创建动画时需要）
* `/settings` 调整周任务和日常任务的频率等
